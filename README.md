## 概要
VELAはPython、PyQtWebEngine製のモダンなウェブブラウザです。

タブブラウジング、ブックマーク、履歴機能、簡易広告ブロッカーなどの、インターネットへの自由なアクセスに必要な機能をすべて押さえた、モダンなクロスプラットフォームブラウザです。

Windowsの場合、setup.exeを起動してください。

## 使い方
ダウンロードしたexeファイルをそのまま起動してください
