VELA 開発マニュアル for Mac

<前提条件>
以下の環境での開発を前提とします
・MacOS Venture またはそれ以降
・実機
・Home brew 4.6.19またはそれ以降
・Python 3.13またはそれ以降

<依存関係の入手>
pip3 python が使えることを確認してください

<開発環境の準備>
python3 -m venv VELADev
source ./VELADev/bin/activate
pip3 install qtawesome PyQt6 PyQt6-WebEngine logzero

<実行方法>
python3 Software.py

<ビルド方法>
pip install pyinstaller
pyinstaller --windowed --onefile --icon=software.ico -F --clean Software.py
