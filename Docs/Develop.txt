開発環境の開き方
0: Python 3.13をセットアップする。
1: env.batを起動し、「VELA Browser Development Kit」に入る
2: PyQt6, qtawesome, PyQt6-WebEngine をインストール(Kitでpipを使うこと)
3: Software.py をKitから実行

ビルド方法
0: Python 3.13をセットアップする。
1: env.batを起動し、「VELA Browser Development Kit」に入る
2: pyinstallerをインストール(Kitでpipを使うこと)
3: pyinstaller --clean --icon=software.ico -F Software.py
4: Kitを閉じ、clean.batを実行
5: buildフォルダを削除
6: distフォルダ内に生成される
