■ 概要

VELAブラウザは、PythonおよびPyQt6, QtWebEngineを用いて開発された、モダンな
Webブラウザです。個人情報の保護、洗練されたUI、本当に必要な機能に特化。

本ブラウザは個人利用および教育研究用途を主に想定しています。

----------------------------------------------------------------------------
■ 主な機能

・ユーザーの個人情報を勝手に送信しない
・垂直タブをネイティブサポート
・Qtベースの洗練されたシステムデザイン
・広告ブロッカー
・開発者ウィンドウ

----------------------------------------------------------------------------
■ 技術仕様

開発言語        : Python 3.13  
UIフレームワーク: PyQt6  
セッション管理  : プロファイル型
