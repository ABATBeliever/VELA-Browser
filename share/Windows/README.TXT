                                   VELA Browser
                         Vital Environment for Liberty Access

====================================================================================================
■ 概要

バージョン     : 1.2.0
対象OS         : Windows 10～
アーキテクチャ : x64

VELA（Vital Environment for Liberty Access）は、PyQt6 と QtWebEngine の技術で開発された、
軽量で拡張性の高い、マルチプラットフォームなウェブブラウザです。
「プライバシー配慮」「移植性」「本当に必要な機能に特化」

■ 主要機能
--------
- Chromium ベースのレンダリング（QtWebEngine）
- タブブラウジング（起動時の復元機能付き）
- ブラウザレベルのトラッキング防止、広告ブロック（EasyListをはじめ、多くの定義ファイルを使用可能）
- プライベートブラウジングモードあり
- UserAgentのランダム化（最新近辺のMicrosoft Edgeに偽装され、起動する度に変化します）
- ブックマーク / 履歴のエクスポート・インポート
- デベロップツールあり
- 水平タブをネイティブサポート

■ 動作環境
----------------
- OS    : Windows 10以降 x64
- ビルド: Pyinstaller または nuitka
