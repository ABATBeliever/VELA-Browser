VELA 開発マニュアル for Linux

<前提条件>
以下の環境での開発を前提とします
Ubuntu 22.04 LTS またはそれ以降
Python 3.10

<依存関係の入手>
sudo apt install python3-pip
sudo apt install python3.10-venv
sudo apt install patchelf

<コードの入手>
git clone https://github.com/ABATBeliever/VELA-Browser

<開発環境の準備>
python3 -m venv VELADev
source ./VELADev/bin/activate
pip install qtawesome PyQt6 PyQt6-WebEngine logzero

<実行方法>
python3 Software.py

<ビルド方法>
pip install pyinstaller
pyinstaller --clean -F Software.py
distフォルダに生成される
